# Supply-Chain-Management-Data-Analysis
This project focuses on analyzing supply chain and inventory data to improve stock management and operational efficiency. The dataset was cleaned and transformed using Excel, and interactive dashboards were created using Power BI and Excel to visualize key metrics. The analysis provides data-driven insights to support demand forecasting, inventory planning, and better decision-making. Data models and relationships were built to ensure accurate reporting and effective supply chain monitoring.

Tools Used
Excel
Power BI
Data Cleaning
Data Modeling
Dashboard Development

Key Objectives
Analyze supply chain and inventory data.
Identify trends in stock and demand.
Improve inventory planning and forecasting.
Create interactive dashboards for insights.
